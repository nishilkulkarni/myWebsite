Projects

An image is worth 16x16 words: Vision Transformers for image recognition at scale
Nishil Kulkarni,

Replicated the Vision Transformer (ViT) model using PyTorch, implementing its core components, including multi-head self-attention and positional embeddings. Exciting dive into self-attention mechanisms and how transformers are shaking up the world of computer vision.

Thoracic Disease Detection in Chest X-Rays Using CNN
Nishil Kulkarni,

Developed a deep learning model for chest X-ray analysis using a DenseNet121 architecture. Integrated a custom sigmoid-activated sequential layer for multi-label classification, enabling the model to take a chest X-ray image as input and output the probability of each thoracic disease. Achieved a mean AUROC of 0.847, outperforming the previous benchmark.

Cross Platform Healthcare App
Nishil Kulkarni,
Paper

Created a cross-platform mobile application using Flutter and Dart, allowing users to input symptoms and receive predictions on the most likely illness, along with the percentage severity of each diagnosis. Designed a custom algorithm to evaluate the likelihood of various illnesses to calculate the similarity between the user's symptom vector and predefined target vectors for multiple conditions .

Connect 4 AI
Nishil Kulkarni,

Developed a Connect 4 AI using the minimax algorithm with alpha-beta pruning, integrating heuristic functions for move ordering, state evaluation, and advanced strategies like column weighting and sequence analysis to optimize decision-making.

Statistical Modeling of Agricultural Derivatives Trading and Its Impact on Global Food Security
Nishil Kulkarni, Dr. R.L Manogna
Paper

Statistical modelling using the panel data regression model, moderating effects model, and panel data regression with threshold variable to analyze financialization due to agricultural commodities. This study seeks to empirically investigate this complicated problem and influence policy choices to ensure a more stable and secure food system by analyzing the role of financialization in global food markets.
